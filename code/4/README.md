Tweet length
=================

Add the character count behavior to the TweetBox

## Presenter Flow

1. Send TweetBox.js using ES6 class to attendees
2. Attendees implement TODOs to add in character count

## Concepts

* ES6 classes
* React controlled components
* React's `setState`
